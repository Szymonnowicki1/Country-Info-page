Country Info Page 🌍

A web application that displays information about countries, including flags, population, region, and capital. Users can search for countries and filter them by region.

The app uses the REST Countries API to fetch country data.

✨ Features

Display a list of countries with:

Flag

Name

Population

Region

Capital

Search for countries by name

Filter countries by region

Alphabetical sorting of countries

Navigation between country details

HashRouter for working navigation on GitHub Pages

API

Country data is provided by the REST Countries API:
https://restcountries.com/v3.1/all

⚠️ Notes

The project is currently not fully responsive – best viewed on desktop screens.

Future improvements could include full responsiveness, additional country details, and integration with a backend.
